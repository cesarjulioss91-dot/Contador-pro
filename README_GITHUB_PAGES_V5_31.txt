NYX Cell Counter Pro v5.31 — GitHub Pages FLAT DIRECT

Objetivo desta versão:
- Corrigir a tela presa em "Carregando módulos...".
- Remover lazy loading bloqueante.
- Desativar service worker/cache antigo que pode segurar versões quebradas.
- Permitir upload pelo celular, com todos os arquivos soltos na raiz do repositório.

Como subir pelo celular:
1. Extraia o ZIP.
2. No GitHub, envie TODOS os arquivos extraídos diretamente na raiz do repositório.
3. Não envie uma pasta fechada; envie os arquivos soltos.
4. Substitua os arquivos antigos quando o GitHub perguntar.
5. Abra: https://SEU_USUARIO.github.io/verificar-arquivos.html
6. Depois abra: https://SEU_USUARIO.github.io/

Se continuar vendo versão antiga:
Chrome Android > Configurações > Configurações do site > Dados armazenados > procure seu domínio github.io > apagar dados.
Depois abra a página novamente.
