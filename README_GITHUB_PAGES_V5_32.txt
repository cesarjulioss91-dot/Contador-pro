NYX v5.32 — GitHub Pages FLAT Cache Killer

Objetivo:
- Corrigir app preso em 'Carregando módulos...' após instalação no Android.
- Remover lazy loader antigo e limpar service workers/caches antigos.

Como subir pelo celular:
1. Extraia o ZIP.
2. Suba TODOS os arquivos diretamente na raiz do repositório.
3. Não precisa criar pastas.
4. Abra primeiro: https://SEU_USUARIO.github.io/limpar-cache.html?v=532
5. Depois abra: https://SEU_USUARIO.github.io/?v=532

Se o PWA instalado continuar abrindo versão antiga:
- Desinstale o ícone do app da tela inicial.
- Chrome > Configurações > Configurações do site > Dados armazenados > apagar dados do domínio.
- Abra limpar-cache.html novamente e reinstale.
