NYX Cell Counter Pro v5.30 — GitHub Pages FLAT Upload

Esta versão foi feita para upload pelo celular.
Não precisa criar pasta assets/ nem icons/.

COMO PUBLICAR PELO CELULAR:
1. Extraia este ZIP.
2. No GitHub, abra o repositório.
3. Toque em Add file > Upload files.
4. Selecione TODOS os arquivos extraídos.
5. Faça Commit changes.
6. Aguarde o GitHub Pages atualizar.
7. Abra verificar-arquivos.html.

A raiz do repositório deve ficar com arquivos como:
- index.html
- 00_style.css
- 00_lazy_bootstrap.js
- 02_script.js
- 10_quiz_bank_v23_data.js
- 19_consultas_rapidas.js
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png

Se o site continuar cru, abra verificar-arquivos.html e veja qual arquivo falhou.
